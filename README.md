# Agatha Christie小说阅读平台前端

## 项目简介
Agatha Christie小说在线阅读平台的前端界面，提供用户友好的小说浏览和阅读体验。

## 技术栈
- HTML
- CSS  
- JavaScript

## 功能特性
- 小说列表展示
- 小说章节浏览
- 在线阅读
- 用户登录/注册界面
- 响应式设计

## 部署方式
- 支持Docker部署
- 静态文件部署

## 安装部署

### Git克隆
bash git clone https://github.com/elizazzzzz/agatha-christie-frontend.git cd agatha-frontend
### 本地运行
直接使用浏览器打开 `index.html` 文件，或使用本地服务器运行。
